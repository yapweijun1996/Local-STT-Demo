POST /api/transcribe Transcribe audio file → text

Content-Type: multipart/form-data

FieldTypeDescription
audiofilerequiredAudio file — mp4 / mp3 / wav / m4a / ogg, max 200 MB
modelstringoptionallarge-v3 (default) · large-v3-turbo · medium · small · base · tiny
languagestringoptionalauto (default) · en · zh · ms · ja · any Whisper language code
enginestringoptionalwhisper-cpp (default, Metal GPU) · faster-whisper (CPU, code-switching)
useGpustringoptional1 (default) · 0 — force CPU (whisper-cpp only)
promptstringoptionalDomain vocabulary hint, e.g. "sales order, SKU A123" — improves proper noun accuracy
 
curl -X POST https://stt.yapweijun1996.com/api/transcribe \
  -F "audio=@recording.wav" \
  -F "model=large-v3" \
  -F "language=auto" \
  -F "engine=whisper-cpp"

import requests

with open("recording.wav", "rb") as f:
    r = requests.post(
        "https://stt.yapweijun1996.com/api/transcribe",
        files={"audio": ("recording.wav", f, "audio/wav")},
        data={"model": "large-v3", "language": "auto"},
    )
print(r.json()["text"])
